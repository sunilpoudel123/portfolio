Simple portfolio showcasing projects, resume, and skills.
Built With

React.js
CSS3
Vercel

Features

Multi-page layout
Responsive design

Getting Started

Clone repo: git clone <your-repo-url>
Install: npm install
Run: npm start
View at: http://localhost:3000

Usage
Edit components in /src/components/ to update content.
